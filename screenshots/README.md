# Official screenshots

Canonical screenshot sets for every Custom RR build type. These are the
source-of-truth copies; the Play-listing copies live under
`fastlane/metadata/android/en-US/images/` and are uploads of the same files.

| Folder | Dimensions | Captured on | Used for |
| --- | --- | --- | --- |
| `phone/` | 1344x2992 (portrait) | Pixel 10 Pro XL emulator | Play phone screenshots, F-Droid `phoneScreenshots` |
| `tablet/` | 1440x2560 (9:16 portrait) | Emulator at tablet size/density | Play 7-inch, 10-inch, and Chromebook slots (all three accept this set) |
| `linux/` | 1920x1080 (landscape) | Linux desktop build, KDE Plasma (Wayland), fullscreen | README, GitHub release page, website |
